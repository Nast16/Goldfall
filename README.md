# Goldfall

Goldfall is a text-based RPG focused on economy, decision-making, and a persistent world.

Instead of farming monsters endlessly, players interact with a dynamic economy where supply, demand, and player actions shape the world.

## Core Features (Planned)

- Dynamic market economy
- Decision-driven gameplay
- Persistent world events
- Text-based command system

## Current State

✅ Basic command system  
✅ Gathering mechanic

## Vision

A living economic RPG simulation built step-by-step.
